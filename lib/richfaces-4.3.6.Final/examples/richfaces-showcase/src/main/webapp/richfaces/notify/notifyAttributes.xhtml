There are lot of attributes which serves for notification's behavior redefinition:

  • stayTime - how long the message will stay after appearing
  • sticky - the message will not disappear, user needs to close it by close button
  • nonblocking - the message will not have closing button, but it will reveal content underneath and user can click through it
  • nonblockingOpacity - the opacity of nonblocking notification
  • showShadow - whenever the shadow should be displayed under the notifications
  • showCloseButton - whenever the close button should be displayed on notifications
